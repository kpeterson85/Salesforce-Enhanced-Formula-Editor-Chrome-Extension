Salesforce Enhanced Formula Editor

A Chrome extension that makes coding and troubleshooting Salesforce formulas quicker and easier.

Install

Features

Editor

Swap out that plain formula textbox with a full code editor that includes syntax highlighting, tabbing, parentheses matching, and find and replace.

More about the Editor
Field Details

Load details about the fields found in the formula like their field type, help text, and their own formulas.

More about Field Details
Smart Formatter

New! Click a button to automatically format the formula in a smart way, keeping it compact and readable where possible.

More about Smart Formatter
